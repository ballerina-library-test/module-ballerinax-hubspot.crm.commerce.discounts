# Seasonal Discount Management

This example demonstrates how to create and manage seasonal discounts in HubSpot's commerce system, including batch creation of promotional discounts for different product categories and retrieving detailed discount information.

## Prerequisites

1. **HubSpot Setup**
   > Refer the [HubSpot setup guide](https://github.com/ballerina-platform/module-ballerinax-hubspot.crm.commerce.discounts#setup-guide) here.

2. For this example, create a `Config.toml` file with your credentials:

```toml
accessToken = "<Your Access Token>"
```

## Run the example

Execute the following command to run the example. The script will print its progress to the console.

```shell
bal run
```
